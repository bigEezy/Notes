# React

Overview: React basics, component model, and common commands.

Key commands:
- Create app: `npx create-react-app myapp` or `npm init vite@latest` for modern setups
- Start dev server: `npm start` or `npm run dev`

Patterns:
- Functional components + hooks
- State: `useState`, effects: `useEffect`

Resources:
- https://reactjs.org/docs/getting-started.html
